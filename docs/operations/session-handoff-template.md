# Session Handoff Template

Use this at the end of any productive AI session — especially when the session
feels heavy, responses are slowing down, or you know you'll need to switch
to a new chat. Fill it out before you close or switch. Save it to the repo.

A new session can restore full operating context in under five minutes by
reading this file plus any linked documents.
