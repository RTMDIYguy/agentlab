## Purpose

Codify the agency values shown in the `Agency Values.jpg` graphic as the
operating culture standard for Uncle Robert Consulting, Agent Lab, Tactix, and
related public-facing brands.

These values are not decorative copy. They define how work should be chosen,
shipped, reviewed, discussed publicly, and improved after mistakes.

## Values

### 1. Servant Leadership

Leadership exists to serve and empower the team rather than command it.
Success is measured by growth, health, autonomy, and the practical capability
created for the people doing the work.

In practice:

- reduce burden before adding complexity
- make others more capable
- lead with service, clarity, and support
- treat authority as stewardship

### 2. Honesty And Integrity

Failures help the agency grow as much as wins when they are owned clearly.
Visible failures increase trust and create the conditions for change.

In practice:

- name what is tested, untested, blocked, or broken
- do not invent results, clients, metrics, or certainty
- turn mistakes into repair logs and better procedures
- keep public lessons useful instead of defensive

### 3. Walk Beside Our Client

The purpose is to help clients become successful, not merely hand them a plan,
template, or software intake. Walking beside the client means helping with
implementation, going the extra mile after the contract ends, and exceeding
expectations through practical support.

In practice:

- translate plans into action
- leave clients more capable than they were before
- clarify handoffs, next steps, and owner responsibilities
- preserve trust after delivery, not only during the sale

### 4. Intentional Excellence

Take ownership of actions and outcomes, individually and as an organization.
Excellence means commitments are fulfilled, mistakes are learned from, and
quality is pursued intentionally.

In practice:

- finish the small controllable thing well
- verify before calling work complete
- document decisions that future agents or humans will need
- improve the system without pretending it is already perfect

### 5. Human Judgment With Responsible Automation

Automation should make client work easier without sacrificing quality,
security, or privacy. Human control stays present at strategic checkpoints so
those pillars remain solid.

In practice:

- automate repeatable work
- keep humans in the judgment loop
- protect keys, secrets, client data, and private context
- review exceptions instead of blindly trusting automation

### 6. Practical Stewardship

Use resources and assets in the most cost-effective way while increasing
impact in simple, useful ways.

In practice:

- prefer low-cost and free routes when they fit the work
- avoid tool sprawl
- use the systems already in place before adding new ones
- spend time, money, and attention where they create real leverage

## Public Use Standard

Public uses should be strategic and tactful. Do not paste the full values block
everywhere.

Use the values in three levels:

| Level | Use |
| --- | --- |
| Full | About page, culture page, onboarding packet, proposal appendix, internal SOPs. |
| Short | Footer line, proposal intro, diagnostic handoff, service pages, LinkedIn about copy. |
| Embedded | Blog posts, case notes, build logs, product pages, and workflow package copy where a specific value explains a decision. |

Short public line:

> Servant leadership, honest build logs, human judgment, responsible
> automation, and practical stewardship.

## Procedure

1. Use the full values only where readers expect culture or trust context.
2. Use one or two values at a time on product, service, or funnel pages.
3. Tie public value language to evidence, such as a proof log, workflow test,
   repair note, or human review point.
4. Do not use values language to make unsupported claims.
5. When a public page references automation, include the human judgment and
   responsible automation principle where appropriate.
6. When a public page references pricing, tools, or implementation choices,
   use practical stewardship language where appropriate.
7. Update this SOP when a value changes or when a new public placement pattern
   is approved.

## Success Criteria

- The agency values are available in the Culture folder and repo.
- Public copy reflects the values without becoming repetitive.
- Agents can cite the values when making editorial, delivery, automation, or
  finance/tooling decisions.
- Client-facing claims remain grounded in tested work and human review.
