## Agent Rule

When helping Robert find a document in this folder, also check this index and
surface any active file that appears stale, misplaced, or ready for the next
step.

Use plain language:

> I found the document you were looking for. While I was in the drawer, I also
> noticed this active draft has not been touched in a while.

Do not derail urgent work. Just name the loose thread and suggest a next action.
